# 🌸 Orchid Continuum  
*A digital Orchid Society database and interactive widget collection*  

## About  
The **Orchid Continuum** is a living digital archive designed to connect orchid enthusiasts, researchers, and conservationists worldwide. It combines:  

- 📖 **Taxonomy & Biodiversity Data** – structured genus and species information  
- 🌍 **Habitat & Ecology Metadata** – native ranges, climate overlays, and conservation insights  
- 📸 **Curated Orchid Photography** – member-submitted and society-archived images  
- 🛠 **Interactive Widgets** – tools for culture tracking, bloom galleries, conservation awareness, and educational outreach  

Built and maintained by **Jeffery Parham** in collaboration with the **Five Cities Orchid Society**, the Orchid Continuum seeks to bring together community knowledge, technology, and conservation.  

---

## Features  
- 🌱 Orchid taxonomy and cultural notes  
- 📊 Integration with biodiversity data (GBIF, Catalogue of Life, etc.)  
- 🌤 Weather and climate comparison widgets for orchid care  
- 🖼 Member photo galleries and digital show tables  
- 🤝 Tools for collaboration between orchid societies worldwide  

---

## How to Use  
- Widgets from this project are being integrated into the **[Five Cities Orchid Society website](https://fivecitiesorchidsociety.org)**  
- Members can test new features during open trials  
- Bug reports and feature suggestions can be submitted via GitHub Issues
